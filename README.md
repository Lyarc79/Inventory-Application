# Inventory App - Game Management
A full-stack web application for catalog and inventory management built with **Node.js**, **Express**, **PostgreSQL**, and **EJS**, following the **MVC architecture**.
## Key Features
- **MVC Architecture:** Structured routes, controllers, and database interaction.
- **Full CRUD:** Complete management for games and categories.
- **Relational DB:** SQL query execution for dynamic data handling.
## Status
> Core backend and database logic are fully operational. Currently refining UI styling.
